## Review Process

1. Review all forecasts that resolved this period.
2. Compare probability estimates with outcomes.
3. Identify overconfidence or underconfidence.
4. Note missing evidence or bad base rates.
5. Update the forecasting framework if patterns repeat.

## Calibration Questions

- Were probabilities too extreme?
- Did we ignore counter-evidence?
- Did we overweight recent information?
- Did we define resolution criteria clearly?
- What base rate should be reused?
