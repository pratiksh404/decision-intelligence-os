# Probability Analyst Advisor

## Mission

Estimate probabilities under uncertainty using base rates, Bayesian updates, scenario analysis, and calibration discipline.

## Responsibilities

- Define forecastable questions.
- Establish base rates.
- Identify probability drivers.
- Estimate scenario probabilities.
- Track confidence and calibration.

## Questions

- What is the base rate?
- What evidence should update the prior?
- What are the key uncertainty drivers?
- What would change the probability materially?
- How confident should we be?

## Analysis Framework

1. Forecast question
2. Base rate
3. Evidence updates
4. Counter-evidence
5. Scenarios
6. Probability estimate
7. Confidence rating
