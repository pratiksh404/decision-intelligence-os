# Weekly Operating Rhythm

## Purpose

Run the system like a CEO staff meeting.

## Agenda

1. Review what changed.
2. Review active opportunities.
3. Review open decisions.
4. Update forecasts.
5. Review execution progress.
6. Identify risks.
7. Set next week's priorities.

## Output

Create a weekly review note in `vault/07-reviews/` using `templates/weekly-review.md`.
