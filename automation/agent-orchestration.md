# Agent Orchestration

## Purpose

Define how AI advisors should collaborate without collapsing into a single generic response.

## Orchestration Sequence

1. Intake the question.
2. Identify artifact type.
3. Gather evidence.
4. Separate facts, assumptions, opinions, predictions, and decisions.
5. Run Intelligence Analyst first.
6. Run CEO, CFO, VC, Skeptic, and Operator independently.
7. Run Probability Analyst after evidence and critiques are visible.
8. Produce final recommendation.
9. Generate execution plan if approved.
10. Set review date.

## Independence Rule

Each advisor should produce analysis before reading the final recommendation. The Skeptic must explicitly seek disconfirming evidence.

## Final Synthesis

The final synthesis must include:

- Executive Summary
- Key Findings
- Advisor Board Analysis
- Probability Assessment
- Recommended Decision
- Execution Plan
- Risks
- Next Actions
- Confidence Assessment
