# Weekly CEO Review Workflow

## Purpose

Create a weekly operating rhythm for the founder to review intelligence, decisions, forecasts, and execution.

## Inputs

- Inbox signals
- Active opportunities
- Open decisions
- Active forecasts
- Execution plans
- Source notes
- Last weekly review

## Review Agenda

1. What changed this week?
2. Which new opportunities emerged?
3. Which decisions need attention?
4. Which forecasts changed?
5. Which execution plans are off-track?
6. What risks increased?
7. What should be the founder's top priorities next week?

## Required Output

- Executive Summary
- Key Findings
- Active Opportunities
- Open Decisions
- Forecast Updates
- Execution Review
- Risks
- Next Week Priorities
- Confidence Assessment

## Output Template

Use `templates/weekly-review.md`.
