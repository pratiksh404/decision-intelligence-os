# Layer 2: Decision Layer
Convert intelligence into disciplined choices. Expose assumptions, record decisions, and forecast probabilities.

## Folders
- `/assumptions/`: Falsifiable premises that underpin decisions. Template: `templates/assumption.md`.
- `/memos/`: Actionable decision memos. Template: `templates/decision-memo.md`.
- `/forecasts/`: Probability forecasting ledger. Template: `templates/forecast.md`.
- `/workflows/`: Logs of decision board reviews.
