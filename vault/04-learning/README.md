# Layer 4: Learning Layer
Audit project outcomes, calibrate forecasts, and identify cognitive biases. This forms the system's compounding feedback loop.

## Folders
- `/outcome-reviews/`: Retrospective post-mortems of resolved decisions. Template: `templates/outcome-review.md`.
- `/calibration/`: Scorecard evaluation and Brier score tracking. Template: `templates/forecast-calibration.md`.
- `/lessons/`: Compounded heuristic rules for future decisions. Template: `templates/lesson.md`.
- `/bias-audits/`: Cognitive bias audits. Template: `templates/bias-audit.md`.
