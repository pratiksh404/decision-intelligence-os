# Review Cadence

## Daily

- Process inbox signals.
- Identify new opportunities.
- Update urgent forecasts.
- Review active execution blockers.
- Capture decisions made outside the system.

## Weekly

- Run Weekly CEO Review.
- Review open decisions.
- Update active forecasts.
- Re-rank opportunities.
- Check execution KPIs.
- Select next week's priority decisions.

## Monthly

- Review forecast calibration.
- Archive stale opportunities.
- Reassess strategic themes.
- Update market intelligence maps.
- Review capital allocation and opportunity cost.

## Quarterly

- Review strategic direction.
- Audit decision quality.
- Evaluate operating system effectiveness.
- Update workflows and templates.
- Define next quarter's strategic priorities.
