# Metadata Schema

## Standard Frontmatter

```yaml
---
title:
type:
status:
created:
updated:
review_date:
owner: Founder/CEO
tags:
  - decision-intelligence
confidence_score:
source_quality:
evidence_strength:
decision_readiness:
related:
---
```

## Recommended Type Values

- opportunity
- forecast
- decision
- intelligence-report
- execution-plan
- source-note
- weekly-review
- principle
- advisor
- workflow

## Recommended Status Values

- draft
- researching
- ready-for-review
- approved
- active
- monitoring
- rejected
- resolved
- archived

## Link Conventions

Use links that describe relationships:

- Related opportunity: `[[Opportunity Name]]`
- Related forecast: `[[Forecast Question]]`
- Related decision: `[[Decision Memo - Topic]]`
- Source note: `[[Source - Publisher - Date]]`
